## 📱 How to Use

1. **Apni details daalo** — Age, Height, Weight
2. **BMI Calculate karo** — Result aur category dekhlo
3. **Goal choose karo** — Muscle Gain / Fat Loss / Maintain
4. **Budget daalo** — Monthly food budget in ₹
5. **Generate dabao** — AI 30-60 sec mein poora plan bana dega
6. **Diet + Workout tabs** mein apna plan dekhlo
7. **Copy karo** aur kisi bhi app mein save karo
